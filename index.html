Started My 8-Bit CPU Project!

Today I officially started my 8-bit CPU project using the Basys 3 FPGA board. My goal is to understand how CPUs work by building one from scratch, including the ALU, control logic, registers, and memory.

I began by researching the architecture and writing out a high-level plan. I'm going to use the onboard block RAM for both instruction and data memory. Next up is designing the ALU and the FSM for instruction execution!
